## Poultry Roasting Guide

| **Poultry Type** | **Ready-to-Cook Weight** (_Lb_)  | **Oven Temp.** (_F_)  | **Approx. Total Roasting Time**  |
|:-----------------|:--------------------------------:|:---------------------:|:---------------------------------|
| **TURKEY**       |              6 - 8               |         325°          | 2-1/2 - 3 hrs                    |
|                  |              8 - 12              |         325°          | 3 - 3-1/2 hrs                    |
|                  |             12 - 16              |         325°          | 3-1/2 - 4 hrs                    |
|                  |             16 - 20              |         325°          | 4 - 4-1/2 hrs                    |
|                  |             20 - 24              |         325°          | 5 - 6 hrs                        |
|                  |                                  |                       |                                  |
| **CHICKEN**      |            2 - 2-1/2             |         400°          | 1 - 1-1/2 hrs                    |
| (_Unstuffed_)    |            2-1/2 - 4             |         400°          | 1-1/2 - 2-1/2 hrs                |
|                  |              4 - 8               |         325°          | 3 - 5 hrs                        |
|                  |                                  |                       |                                  |
| **DUCK**         |              3 - 5               |         325°          | 2-1/2 - 3 hrs                    |
| (_Unstuffed_)    |                                  |                       |                                  |
